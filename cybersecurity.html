Learning Roadmap

Your 6-Month Journey

Phase 1 — Months 1-2

IT & Networking Fundamentals

OSI model, TCP/IP, DNS, DHCP, networking protocols, Linux/Windows administration, virtual machines setup.

Phase 2 — Month 2-3

Security Concepts & Cryptography

CIA triad, threat models, encryption algorithms, PKI, SSL/TLS, firewalls, IDS/IPS, VPNs.

Phase 3 — Months 3-4

Ethical Hacking & Pen Testing

Footprinting, scanning, enumeration, exploitation, post-exploitation using Kali Linux, Metasploit, Nmap, Burp Suite.

Phase 4 — Month 4-5

Web App & Cloud Security

OWASP Top 10, SQL injection, XSS, CSRF, API security, AWS/Azure security architecture and cloud pen testing.

Phase 5 — Month 5

SOC Operations & SIEM

SOC workflows, Splunk, IBM QRadar, incident response, threat intelligence, log analysis and security monitoring.

Phase 6 — Month 6

Capstone Project & Placement

Real-world penetration testing project, digital forensics, resume polish, mock interviews, and placement support.
